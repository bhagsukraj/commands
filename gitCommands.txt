1.How to initialize git in the folder you are working
$ git init

2.How to sync central repository with local repository by adding origin
$ git remote add origin 'https://github.com/bhagsukraj/pythonMysql.git'

3.How to pull files from central repository?
$ git pull origin master

4.How to get the status of files in git?
$ git status

5.How to add files to the index?
$ git add gitCommands.txt

6.How to commit files to the github(-a for committin all files, -m to provide mandatory msg)
$ git commit -a -m 'commit complete'

7.How to add all files in one shot to the index(-A indicates all files)
$ git add -A

8.How to create new branch under master
$ git branch firstbranch

9.How to go into the branch
$ git checkout firstbranch
