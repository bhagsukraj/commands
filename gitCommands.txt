1.How to initialize git in the folder you are working?
$ git init

1.1.How to undo git from the folder.
$ rm -rf .git

2.How to sync central repository with local repository by adding origin?
$ git remote add origin 'https://github.com/bhagsukraj/pythonMysql.git'

3.1.How to pull files from central repository?
$ git pull origin master

3.2.How to clone a folder to the current directory?
$ git clone https://github.com/bhagsukraj/gitTests.git

3.3.How to get remote repository location
$ git remote -v

4.How to get the status of files in git?
$ git status

5.How to add files to the stagin area or index?
$ git add gitCommands.txt

5.1.How to remove files from staging area?
$ git reset gitTest3.txt

5.2.How to add all files in one shot to the index?
$ git add -A
(-A indicates all files)

6.How to commit files to the git repository?
$ git commit -a -m 'commit complete'
(-a for committin all files, -m to provide mandatory msg)

7.1.How to create new branch under master?
$ git branch firstbranch

7.2.How to go into the branch?
$ git checkout firstbranch

7.3.How to see all the branches for the folder?
$ git branch -a

7.4.How to delete a branch if you are sure?
$ git branch -D firstbranch

7.5.How to delete a remote branch?
$ git push origin --delete firstbranch

8.1.How to create public ssh key?
$ ssh-keygen
After key generation paste it in SSH keys in github

8.2.How to activate the SSH key in the git hub?
$ ssh -T git@github.com

9.How to push code to github(firstbranch suggests to create a new branch)
$ git push origin firstbranch

9.1.How to make a certain path default for default push and pull
$ git push -u origin master(After this statement the below statements can be run)
$ git push
$ git pull

10.1.How to check git config
$ git config --list

10.2.Git config lines for meld
git config --global diff.tool meld
git config --global difftool.meld.path "/usr/bin/meld"
git config --global difftool.prompt false

git config --global merge.tool meld
git config --global mergetool.meld.path "/usr/bin/meld"
git config --global mergetool.prompt false

10.3.How to launch a diff tool comparing origin/master and current branch?
$ git difftool origin/master

10.4.How to launch a diff tool comparing head and current branch?
$ git difftool master

10.5.How to launch a diff tool comparing comparing committed and uncommitted
changes of any branch ?
$ git difftool -t meld

10.6.How to check the global and local config
$ git config --global -l
$ git config --local -l

11.How to ignore some files.
$ touch .gitignore -- (Creates gitignore file that will contain patterns)

12.How to get the status after git commit?
$ git stat

13.How to move the commits on one branch to another brach or cherry-pick?
$ git cherry-pick 9d10fc70
the last arguments is the part of a string from the commited instance.

14.How to clean all the untracked files from the working directory?
$ git clean -df

15.How to get know the exact sequence of stuff that i have been doing?
$ git reflog

16.1.How to stash uncommitted changes?
$ git stash save 'stashing changes'
save 'stashing changes' will save the stash with that comment

16.2.How to check the stash list?
$ git stash list

16.3.How to get back the stashed stuff?
$ git stash apply stash@{0}
'stash@{0}' is obtained from the git stash list command

16.4.How to clear the git stash list?
$ git stash clear
