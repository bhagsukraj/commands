1.How to initialize git in the folder you are working?
$ git init

1.1.How to undo git from the folder.
$ rm -rf .git

2.How to sync central repository with local repository by adding origin?
$ git remote add origin 'https://github.com/bhagsukraj/pythonMysql.git'

3.1.How to pull files from central repository?
$ git pull origin master

3.2.How to clone a folder to the current directory?
$ git clone https://github.com/bhagsukraj/gitTests.git

3.3.How to get remote repository location
$ git remote -v

4.How to get the status of files in git?
$ git status

5.How to add files to the stagin area or index?
$ git add gitCommands.txt

5.1.How to remove files from staging area?
$ git reset gitTest3.txt

5.2.How to add all files in one shot to the index?
$ git add -A
(-A indicates all files)

6.How to commit files to the git repository?
$ git commit -a -m 'commit complete'
(-a for committin all files, -m to provide mandatory msg)

7.1.How to create new branch under master?
$ git branch firstbranch

7.2.How to go into the branch?
$ git checkout firstbranch

7.3.How to see all the branches for the folder?
$ git branch -a

8.1.How to create public ssh key?
$ ssh-keygen
After key generation paste it in SSH keys in github

8.2.How to activate the SSH key in the git hub?
$ ssh -T git@github.com

9.How to push code to github(firstbranch suggests to create a new branch)
$ git push origin firstbranch

9.1.How to make a certain path default for default push and pull
$ git push -u origin master(After this statement the below statements can be run)
$ git push
$ git pull

10.How to check git config
$ git config --list

11.How to ignore some files.
$ touch .gitignore -- (Creates gitignore file that will contain patterns)
