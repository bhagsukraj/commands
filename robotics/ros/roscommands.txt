Setup

$ sudo apt-get install catkin
$ source /opt/ros/kinetic/setup.bash -- Setup ros kinetic
$ mkdir -p ~/catkin_ws/src -- create catkin workspace directory
$ cd ~/catkin_ws/src -- navigate into the workspace directory
$ catkin_init_workspace -- command to create CMakeLists.txt file
$ cd ~/catkin_ws -- navigate to the workspace
$ catkin_make -- create build and level/build for c++ prog/level has files for setup config
$ source devel/setup.bash --run setup
$ cd ~/catkin_ws/src -- navigate to src folder
$ catkin_create_pkg myfirstcode rospy -- create new folder myfirstcode/depends on rospy/
                                         /create CMakeLists.txt package.xml

roscommands

$ cd /opt/ros/kinetic/share/rospy_tutorials
$ roscore -- to initiate ros.
$ rosrun rospy_tutorials talker -- runs the starter node
$ rosrun rospy_tutorials listener -- runs the listener node
$ rqt_graph -- to display the diagram linking talker/listener
$ rostopic list -- displays the list of ros topics
$ source devel/setup.bash -- runs the setup file of the catkin workspacec
$ chmod u+x 1topic_publisher.py -- set executable permissions for the file
$ rosrun myfirstcode 1topic_publisher.py -- Run the file as ros node
$ rostopic list -- List out all current topics
$ rostopic echo counter -n 5 -- echo's out the values of the topic counter upto 5 no's
$ rostopic info counter -- give information about the counter
